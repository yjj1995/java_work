### 垃圾回收
* 不再使用的内存应回收

* C， C++ 由程序员手动编写代码回收 （优点：能够在内存不使用的时候快速回收，缺点：容易出现BUG）

* java 垃圾回收是自动的，开了一个系统级线程自动去检测哪些内存不用了然后回收掉（优点：自动的，缺点：回收不及时）
